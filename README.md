
# 📊 iM뱅크 금융데이터 기반 ML & DL 분석 실습

본 레포지토리는 **iM DiGital Banker Academy** 과정 중 진행한  
금융 데이터 기반 **머신러닝(ML) · 딥러닝(DL) 분석 실습 노트북 모음**입니다.

단순 모델 구현이 아니라,  
👉 **금융 데이터의 시간 구조 이해 → 전략 로직 설계 → 모델 적용 → 백테스트 흐름**까지  
실무 관점에서 단계적으로 학습한 내용을 정리했습니다.

## 🧩 주요 학습 내용

### 1️⃣ 금융 시계열 데이터 이해
- `DatetimeIndex`, `TimedeltaIndex` 구조 이해
- 금융 데이터에서 시간 축이 중요한 이유
- Pandas 기반 시계열 연산 실습

### 2️⃣ 백테스트(Backtesting) 구조
- 전략 수익률 계산 방식
- 누적 수익률 / Drawdown 개념
- 단순 수익 비교가 아닌 **전략 검증 관점** 학습

### 3️⃣ 전략 로직 구현
- 모멘텀 전략 (Momentum Strategy)
- 단기 가치 전략 (Short-term Value)
- 전략 조건 → 코드 로직 변환 연습

### 4️⃣ ML · DL 전략 확장
- 기존 규칙 기반 전략 위에 딥러닝 모델 적용
- DL 결과를 **전략 필터/보조 지표로 활용**
- 모델 예측을 그대로 쓰지 않고 **금융 해석 관점에서 결합**
